# 📓 The Rambler – 30-Day MVP Plan

**Goal**: Deliver a calm, trustworthy place where users can dump unstructured thoughts and receive gentle, useful structure in return.

## 🎯 MVP Success Criteria
- Write freely without friction.
- Ask for structure when they want it.
- Trust that nothing happens without consent.
- Leave feeling lighter or clearer.

## 🧱 Feature Set (Strictly Limited)
### 1. ✍️ Capture
- Single-page writing surface.
- No formatting beyond paragraph breaks.
- Autosave on pause or blur.

### 2. 🧩 On-Demand Processing
- **AI Engine**: Google Gemini 2.5 Flash.
- One explicit action: “Process this ramble”.
- **Backend Flow**: Take user input → Feed to Gemini API → Return summarized results → **Record in backend database**.
- **Outputs**: Short summary ("Kernels of knowledge"), bullet list of ideas, questions detected, topics mentioned.
- Disclaimer: “This is an interpretation, not a conclusion”.

### 3. 📚 History (Minimal)
- Simple chronological list (date, first line, word count).

## 🗓️ 4-Week Build Roadmap
### Week 1: Foundation
- Writing surface & Autosave.
- History list.
### Week 2: Processing Core
- Processing trigger & schema.
- Neutral tone enforcement.
### Week 3: Trust & Polish
- Delete flow.
- UI tuning for "quiet" feel.
### Week 4: Launch Readiness
- Processing limits.
- Simple onboarding.

---

## 🚫 Out of Scope
- No tasks, goals, or reminders.
- No notifications or streaks.
- No AI "coaching" or gamification.
